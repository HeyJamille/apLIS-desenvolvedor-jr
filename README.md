## Rodar Projeto
- PHP: PHP -S localhost:8000 -t public public/index.php
- NodeJS: npm start
- React: npm start

## Instruções
- 1º Inicie o seu servidor MYSQL localhost, eu estou usando o XAMPP
- 2º Ligue o MYSQL e APACHE no XAMPP
- 3º Rode o projeto em PHP para criar o banco de dados hospital_db e a tabela de médicos
- 4º Rode o projeto em NodeJS para criar a tabela de pacientes
- 5º Rode o front em REACT
- 6º Tudo certo para começar a testar o CRUD completo do projeto no front
